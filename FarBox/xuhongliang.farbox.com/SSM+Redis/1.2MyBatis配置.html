typeAliases 别名

定义别名简写来代表类，分为系统定义别名和自定义别名。

自定义别名

说明：

  • typeAlias 中 alias 对应别名，type 对应类，每次只能定义一个类的别名
  • package 中 name 中的包会被扫描，内部的 POJO 会自动定义别名，规则为字母变为小写
  • 当有重名现象时，用注解重新命名@Alias("user3")

typeHandler 类型转换器

分为 jdbcType 定义数据库类型和 javaType 定义 Java 类型。
作用为承担他们之间的相互转换。
很多情况下我们不需要去配置他们，对于需要使用自定义枚举的场景，或者数据库使用特殊数据类型的场景，可以自定义 typeHandler 去处理类型之间的转换问题。
