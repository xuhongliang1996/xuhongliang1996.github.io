知道元素的高度

不知道网页布局中的高度这是很常见的；有很多原因：

  • 如果宽度改变文本流可以改变高度
  • 文本样式中的差可以更改高度
  • 文本字号的变化可以改变高度
  • 具有固定宽高比的元素（如图像）可以在调整大小时更改高度等等。
