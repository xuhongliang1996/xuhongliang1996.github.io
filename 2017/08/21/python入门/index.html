使用list和tuple

list

list 是一种有序的集合，可以随时添加和删除其中的元素。

用len()函数可以获得list元素的个数：

用索引来访问list中每一个位置的元素，超出范围报错，访问最后一个元素的索引是classmates[len(classmates) - 1]。
索引为负数时如-1，代表从倒数第一个。

append()向list中追加元素到末尾：classmates.append('Adam')

insert()向list中插入元素到指定位置：classmates.insert(1, 'Jack')

pop()删除list末尾的元素：classmates.pop()

pop(index)删除list指定位置上的元素：classmates.pop(1)

dict

dict全称dictionary，在其他语言中也称为map，使用键-值（key-value）存储，具有极快的查找速度。

dict内部存放的顺序和key放入的顺序是没有关系的，dict的key必须是不可变对象，字符串、整数等都是不可变的，list是可变的，就不能作为key。

dict 与 list 对比

和list比较，dict有以下几个特点：

  1. 查找和插入的速度极快，不会随着key的增加而变慢；
  2. 需要占用大量的内存，内存浪费多。

而list相反：

  1. 查找和插入的时间随着元素的增加而增加；
  2. 占用空间小，浪费内存很少。

所以，dict是用空间来换取时间的一种方法。
